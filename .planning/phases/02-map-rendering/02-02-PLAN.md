<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| VBA -> OS Shell | GenerateMap() calls Shell with a file path; path is derived from ThisWorkbook.Path (trusted) + fixed constant OUTPUT_FILE |
| OS Shell -> Browser | explorer.exe opens the HTML file; browser sandbox isolates local file execution |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-03 | Tampering | Shell path injection via ThisWorkbook.Path containing special characters | accept | ThisWorkbook.Path is controlled by Excel (the user's save location) — not user-supplied input. Triple-quote wrapping (`"""` & filePath & `"""`) handles spaces. Parentheses and brackets in paths may cause Shell errors on some Windows configs (A4 assumption). Disposition: accept — user controls the save location; document as a known limitation if they see Shell errors. |
| T-02-04 | Elevation of Privilege | GenerateMap() runs under the workbook's macro trust level | accept | User already granted macro execution trust by enabling macros. No privilege escalation beyond what is already granted. |
| T-02-SC | Tampering | CDN assets loaded at browser open time | mitigate | Leaflet 1.9.4 loaded with SRI integrity hash. MarkerCluster 1.5.3 pinned by version in URL. Both packages are established (10+ years, millions of weekly downloads per research audit). |
</threat_model>
